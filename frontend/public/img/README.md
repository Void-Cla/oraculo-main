# Imagens do Painel

## Papel

Concentrar os ativos visuais estaticos usados pelo frontend publico do Oraculo.

## Arquivos esperados

- `oraculo.png`: icone principal usado como favicon e identidade visual do dashboard.

## Razao logica

Separar imagens de HTML, CSS e JavaScript melhora organizacao e evita duplicacao de caminhos visuais espalhados pelo frontend.

## Regras de uso

- os arquivos desta pasta devem ser leves e prontos para entrega direta pelo servidor estatico;
- nomes devem ser estaveis para nao quebrar referencias do painel;
- quando um ativo visual fizer parte da identidade da Void-Hub, ele deve ser centralizado aqui para reaproveitamento consistente.
